# Live Event Registration & Automated Raffle System 🎟️🚗

A high-speed, iPad-optimized web application deployed via Firebase, built specifically to handle live customer registration and automated winner selection for a high-stakes corporate exhibition car raffle.

## 🛠️ Tech Stack
* **Environment:** Web Application (Mobile/iPad Optimized)
* **Backend & Database:** Python & Firebase
* **Architecture:** Cloud deployment with secure local data export capabilities

## ✨ Core Features
* **High-Speed iPad Registration:** Designed with a frictionless UI to allow event staff to process massive crowds and input coupon numbers rapidly.
* **Strict Data Integrity:** Engineered real-time database validation to automatically block duplicate Name or ID entries, ensuring absolute fairness in the draw.
* **Automated Winner Draw:** Built a randomized, unbiased withdrawal algorithm that securely queries the database to announce the winner on the spot.
* **Secure Lead Generation:** Architected the database to securely store and export customer data for post-event promotional campaigns and security auditing.

## 🧠 The Build & Use Case
**The Challenge:** Live exhibitions move fast. We needed a system that wouldn't crash under pressure, could be accessed instantly by multiple staff members on different iPads, and completely eliminated the risk of human error or cheating (like someone registering the same coupon twice).

**The Solution:** I built this system and hosted it on Firebase for instant, device-agnostic access. The automatic duplicate-catching error system saved immense time at the registration desk, and the automated draw feature provided a professional, transparent experience for the attendees.
